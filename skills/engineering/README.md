# Engineering Skills

日常代码工作使用的 skills。

- [diagnose](./diagnose/SKILL.md) — 面向困难 bug 和性能回退的纪律化诊断循环。
- [domain-model](./domain-model/SKILL.md) — 对照现有 domain model 盘问计划并更新 `CONTEXT.md` / ADR。
- [github-triage](./github-triage/SKILL.md) — 通过 label-based state machine 分诊 GitHub issues。
- [improve-codebase-architecture](./improve-codebase-architecture/SKILL.md) — 发现 codebase 中的 deepening 机会。
- [tdd](./tdd/SKILL.md) — 使用 red-green-refactor 循环做 test-driven development。
- [to-issues](./to-issues/SKILL.md) — 将计划、spec 或 PRD 拆成 GitHub issues。
- [to-prd](./to-prd/SKILL.md) — 将当前上下文整理成 PRD 并提交为 GitHub issue。
- [zoom-out](./zoom-out/SKILL.md) — 请求更高层次的代码上下文。
