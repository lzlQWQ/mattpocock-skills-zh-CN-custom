## What it does

`writing-great-skills` 是编写和编辑优秀 skills 的 reference：它提供词汇、原则和判断标准，让 skill 可预测、可调用、可维护。

它不是通用写作建议。它关注 agent skills 的结构、trigger、progressive disclosure 和行为边界。

## When to reach for it

你通过输入 `/writing-great-skills` 调用它；agent 不会自行触发。

当你要创建、重写或 review 一个 skill，并需要判断它是否清晰、可触发、不过度膨胀时使用它。

## Where it fits

它是 standalone reference，也支撑 skill 作者维护本仓库。完整地图见 [ask-matt](https://aihero.dev/skills-ask-matt)。
